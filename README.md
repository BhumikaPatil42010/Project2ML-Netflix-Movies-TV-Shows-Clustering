## Project Overview
This project focuses on clustering Netflix Movies and TV Shows using Machine Learning techniques.  
The goal is to group similar content based on features such as genre, duration, rating, and country to discover hidden patterns in Netflix data.

This project is developed as part of a Machine Learning & Data Analytics learning project.

---

##  Objective
- Perform data preprocessing and cleaning
- Conduct Exploratory Data Analysis (EDA)
- Apply Feature Engineering techniques
- Implement clustering algorithms
- Identify similar groups of Netflix content

---

##  Dataset Information
The dataset contains information about Netflix Movies and TV Shows including:
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
